교육활동
2021. 03월 2021년 온택트 디자인큐레이터 육성사업 (참여) 대전 디자인 진흥원 주관
온택트 디자인큐레이터 육성사업을 통해 여러 디자인과 접하며 디자인을 좀 더 폭넓게 알 수 있었고 팀원들과 사업 기획안을 짜는 등의 실습을 통해 기획능력도 올릴 수 있었습니다. 무엇보다 현장에서 일하고 계신 분들의 이야기를 직접 듣고 배울 수 있는 좋은 경험을 할 수 있었습니다.
인턴활동
2021. 06월(주) 한밭아이오티 인턴 (6개월 반)
한밭아이오티에서 인턴을 경험하면서 현장에서의 실무 경험을 얻을 수 있었고 처음 해보는 일도 시행착오를 겪어가며 스스로 노력해 극복함으로써 자신이 성장했음을 느낄 수 있었습니다.

Main Screen
  • Main Page 각 콘텐츠의 연결성을 부여하여 자연스럽게 시선을 아래로 유도함으로써 셀렉토의 브랜드 가치관과 내용을 알기 쉽게 전달하도록 제작하였습니다.
  • Scrolling Page 자연스러운 시각적 흐름 텍스트의 효율적인 시각 흐름을 고려하여 각 콘텐츠의 목적을 스크롤 페이지에 배치하였습니다
  • Main Banner 자연스러운 시각적 흐름 자신의 이미지한 컨셉을 잘 보여줄수있게 제작하였습니다.
  • Web Portfolio 포트폴리오를 한 눈에 웹 포트폴리오를 종류별로 보기 쉽게 정리하고 가로 스크롤 활용하여 나중에 자료를 추가하게 되어도 형태를 무너지지 않고 쉽게 추가 할 수 있게 만들었습니다.
  • Portfolio 1 이벤트를 한 눈에 간단한 설명과 함께 사진을 중심으로 보기 쉽게 정리하였고, 가로 스크롤 활용하여 나중에 자료를 추가하게 되어도 형태를 무너지지 않고 쉽게 추가 할 수 있게 만들었습니다.
  • Portfolio 2 눈에 잘 들어오는 크기 저작권이 있는 회사에서 만든자료는 따로 분리하여 제작하였습니다.
    간단한 설명과 함께 사진을 중심으로 보기 쉽게 정리하였고, 가로 스크롤 활용하여 나중에 자료를 추가하게 되어도 형태를 무너지지 않고 쉽게 추가 할 수 있게 만들었습니다.
